# OOD-Final-Project

To run this program you need to run the CClient.java file, if using for multiplayer you can update the local ip (line 90-100 in ChatClient.java) to your local ip or local host
If playing multiplayer or using the chatrooms, you must first run the CServer.java file then run CClient.java, after that the server will meed you to register a new account the first login, and then you will
be able to log in to the application using that username and password.
